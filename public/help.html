VDraw 帮助

  • 快捷键
    • Shift + 滚轮 平移画布
    • Alt + 滚轮 缩放画布
    • Ctrl + = 放大画布5%
    • Ctrl + Shift + = 放大画布50%
    • Ctrl + - 缩小画布5%
    • Ctrl + Shift + - 缩小画布50%
    • Ctrl + 0 恢复画布
    • B 刷子
    • P 画笔
    • E 擦除
    • A 选择
    • Ctrl + Z 撤销
    • Ctrl + Shift + Z 重做
    • Ctrl + S 保存 SVG
    • F1 帮助
    • Ctrl + T 切换工具栏显示
    • Ctrl + Shift + N 新建图层
  • 多人合作
    • 房间名称必须为纯字母（大小写均可）
    • 房间如果不存在，自动创建房间；如果房间存在则校验密码
    • 密码允许为空
    • 空房间将被删除，请注意保存。

  • 保存功能还在完善
